Introducción a Beamer

pbaeyens 14-03-2015

beamer es una clase de documento de LaTeX que nos permite crear diapositivas en pdf y otros formatos. Permite el uso de transiciones, mostrar elementos selectivamente en la pantalla y aprovechar las capacidades de tikz, pgf y la mayoría de paquetes de LaTeX para personalizar el aspecto de la presentación al detalle.

Puedes encontrar una guía básica escrita en beamer aquí. No he podido cubrir todo, así que otros recursos que puedes utilizar son:

  • La guía oficial de beamer
  • Presentations Wikibooks
  • TeX Stack Exchange

Además, puedes utilizar las plantillas disponibles en M42/plantillas para facilitar tu trabajo.
