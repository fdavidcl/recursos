--- title: Nuevo recurso layout: post ---

Necesitas una cuenta de GitHub para enviarnos recursos.

Paso a paso

Si tienes acceso al repositorio

Tras pulsar Enviar haz clic en Commit new file.

Si no tienes acceso de escritura

Hemos tratado de que este proceso sea lo más simple posible. Únicamente, pulsa Enviar y sigue los botones verdes: Primero, confirma que quieres hacer un fork del repositorio.

Después, propón el nuevo archivo (los datos estarán rellenos, no tienes que hacer nada más).

Finalmente, crea la pull request.
